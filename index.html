USDT.z

Experimental USD-Referenced Token on BNB Smart Chain

About the Project

USDT.z is an experimental BEP-20 token deployed on the BNB Smart Chain. The project is designed for research, development, and testing purposes within decentralized finance (DeFi) environments.

The token uses an upgradable proxy architecture (ERC1967 / UUPS), allowing controlled upgrades of logic while maintaining the same on-chain address.

USDT.z is not affiliated with Tether, USD₮, or any centralized stablecoin issuer and does not represent a claim on fiat currency.

Project Status

USDT.z is an experimental and educational project. It is not actively marketed and exists primarily for architectural testing, learning, and on-chain experimentation.
